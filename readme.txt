Plugin para instalação da TAG da Cazamba em um site/blog Wordpress, sem a necessidade de editar o código fonte.

== Description ==

Através deste plugin é possível a instalação da tag da Cazamba em seu site sem a necessidade de alterar o código fonte ou experiência avançada.

Acesse através do nosso painel a área de implementação de TAG e obtenha o seu Token. Através do painel administrativo do Wordpress vo ê encontrará a opção "Cazamba", ao entrar nela basta informar seu "token" que a tag já estará instalada em seu site.

== Installation ==

1. Faça o upload dos arquivos para o diretório `/wp-content/plugins/`
2. Ative o plugin através do link 'Plugins' no painel administrativo do WordPress
3. Informe seu Token (Obtido pelo painel de Publishers/Cazamba em seu painel administrativo do Wordpress)
